Import the Data and Wrangle it into the required format

This includes making date objects, data type coersion, and a join

RF.df=read.csv("https://github.com/Bio724D/Bio724D_2023_2024/raw/main/data/RF_Data_for_Class.csv") |> 
  mutate(Date=make_date(Year,Month,Day))|>  #Make a date object using the make_date function
  mutate(RF=as.numeric(RF)) #Coerce the RF column to be numeric

df=read.csv("https://github.com/Bio724D/Bio724D_2023_2024/raw/main/data/Gray_Whale_Strandings_For_Class_MODIFIED.csv") |> 
  mutate(Date=mdy(Date)) |> #Make a date object using the mdy() function
  left_join(RF.df)  #Use a left join to the the RF on the days the whales stranded
## Joining with `by = join_by(Date)`
